Project Overview:
- The project is a web application for Thanga Mani, an e-commerce site for peanut burfi.
- It uses Razorpay for payments, Firebase for data storage, and email notifications.
- Backend is PHP-based for compatibility with Hostinger shared hosting.

Deployment Instructions:
========================
For Local Development (without XAMPP):
1. Ensure PHP 8.1+ is installed (run `php -v` to check).
2. Run `composer install` in project root to install dependencies.
3. Run `php -S localhost:8000` in project root.
4. Access at http://localhost:8000
5. The server is now running - test the payment flow.

For Hostinger:
1. Upload all files to public_html or subdomain folder.
2. Ensure PHP 8.1+ is enabled in Hostinger control panel.
3. Update config/config.php with production credentials:
   - Change Razorpay keys to live keys
   - Update SMTP settings if needed
   - Set correct admin email
4. Test payment flow.

Notes:
======
- PHP backend handles payments, emails, and data storage.
- Orders are saved to logs/orders.jsonl file.
- For production, update Razorpay keys to live, and secure SMTP credentials.
